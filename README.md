# 🌤️ SkyCast - Premium Weather Experience

SkyCast is a modern, responsive weather application that provides real-time weather data and 6-day forecasts with a premium glassmorphic design.

![SkyCast Preview](https://raw.githubusercontent.com/jamuna-hub/skycast-weather-app/main/preview-placeholder.png)

## ✨ Features

- **Global Search**: Get weather data for any city or country.
- **Real-time Data**: Current temperature, humidity, wind speed, and visibility.
- **6-Day Forecast**: Plan your week with accurate daily forecasts.
- **Dynamic UI**: Backgrounds change dynamically based on the current weather condition.
- **Glassmorphism**: A sleek, modern aesthetic with blurred elements and smooth animations.
- **No API Key Required**: Uses the Open-Meteo API for hassle-free integration.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Custom design tokens and glassmorphism.
- **Modern JavaScript**: ES6+, Fetch API, Lucide Icons.
- **Open-Meteo API**: Professional weather and geocoding data.

## 📖 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/jamuna-hub/skycast-weather-app.git
   ```
2. Open `index.html` in your favorite browser.
3. Enjoy the premium weather experience!
